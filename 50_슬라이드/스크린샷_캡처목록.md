# 강의 슬라이드 — SAP 화면 캡처 목록

> 슬라이드의 회색 점선 상자(📷 자리표시자)에 들어갈 SAP 화면 캡처 목록.
> 대상 S/4HANA 2023 시스템 준비 시(`40_패키지/시스템준비/04_강의전_준비체크리스트.md` 수행 시) 함께 캡처한다.

## 캡처 규칙

- 강사 슬롯 **00번**(`ZEDU00_*`) 객체 기준으로 캡처한다 — 슬라이드의 `##` 표기와 어긋나지 않도록, 캡처에 학생 번호가 보여도 무방하다는 점을 OT에서 한 번 언급.
- SAP GUI 테마·폰트는 강의장에서 실제 사용할 설정으로 통일한다 (프로젝터 확인 겸용).
- 해상도: 화면 전체가 아니라 **해당 영역 위주로** 자른다. 최소 가로 1200px 권장.
- 저장 위치·이름: `50_슬라이드/captures/D1-01.png` 형식 (ID와 파일명 일치).
- 개인정보 주의: 사용자 ID·서버 주소가 노출되는 화면은 필요 시 흐림 처리.

## 슬라이드에 넣는 방법

1. 캡처를 `50_슬라이드/captures/`에 저장
2. PowerPoint에서 해당 슬라이드의 점선 상자 위치에 그림 삽입 → 점선 상자 삭제
3. 상자 안 캡션(예: "Fields 탭 — 9필드 입력 완료 화면")과 같은 내용이 보이는지 확인

## Day 2 (SLIDES_Day2.pptx) — 2컷

슬라이드에 실제 배치된 📷 자리표시자는 아래 2컷이다(회색 점선 상자). Day 2는 개념·코드 중심이라 캡처 지점을 선택화면·ALV 두 결정적 순간으로 압축했다.

| ID | 사용 슬라이드 | T-code / 화면 | 캡처에 반드시 보여야 하는 것 | 상태 |
|----|---|---|---|:---:|
| D2-01 | 37 (U13 선택 텍스트·실행 관찰) | ZEDU00_U13 실행 후 선택화면 | SELECT-OPTIONS 주문번호 화살표 → 복수선택 팝업(포함/제외 탭). 선택 텍스트(상태/주문일/주문번호)가 라벨로 보일 것 | ☐ |
| D2-02 | 53 (U15 실행 후 관찰) | ZEDU00_U15 실행 결과 ALV | ALV 기본 화면 — 컬럼 제목이 한글(주문번호/고객명/제품명/금액 등) + AMOUNT 컬럼 합계(Σ) + 상단 툴바(정렬·필터·엑셀) | ☐ |

> 라이브로 함께 보여줄 만한 화면(SE38 WRITE 실행 결과, SE16N 조건 조회, F4 도메인 고정값 목록)은 슬라이드 자리표시자로 넣지 않았다 — 강사 시연으로 대체한다. 필요 시 이 절에 ID를 추가하고 해당 슬라이드에 `shot()` 자리표시자를 넣는다.
