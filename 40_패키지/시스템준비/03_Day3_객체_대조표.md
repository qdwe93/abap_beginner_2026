## 메시지 클래스 `ZEDU##_MSG`

| 번호 | 텍스트 |
|------|--------|
| 001 | 주문 &1이(가) 등록되었습니다. |
| 002 | 주문 &1이(가) 수정되었습니다. |
| 003 | 주문 &1이(가) 삭제되었습니다. |
| 010 | 존재하지 않는 고객입니다: &1 |
| 011 | 존재하지 않는 제품입니다: &1 |
| 012 | 수량은 1 이상이어야 합니다. |
| 013 | 입력할 수 없는 상태값입니다: &1 |
| 020 | 완료된 주문은 수정할 수 없습니다. |
| 021 | 신규 상태의 주문만 삭제할 수 있습니다. |
| 022 | 처리할 행에 커서를 두고 버튼을 누르세요. |

메시지 023은 만들지 않는다. X 주문 수정 차단은 다음 리터럴을 사용한다.

```abap
MESSAGE '취소된 주문은 수정할 수 없습니다.' TYPE 'S' DISPLAY LIKE 'E'.
```

- ZEDU00_S_LIST
